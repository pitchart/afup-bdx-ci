L'existant





  • Environnement de développement hétérogène

Avantages

  • Gitlab + CI
  • Plusieurs versions de PHP disponibles 5.6 -> 7.1
  • Multiples pipelines (branche / version / steps)
  • Multiples tâches (test / deploy / static analysis / doc)

Inconvénients

  • Environnements d'exécution hétérogènes
  • Faux positifs ou erreurs liées uniquement à la CI
  • Conflits de versions PHP
  • Gestion des fixtures (SQL)
  • Assets

Ressentis

  • Feedback lent
  • Goulots d'étranglement au sein de l'équipe (compétences non mutualisées)
  • Temps de première contribution long (besoin d'un validateur)
  • Adhésion limitée (manque de confiance dans les résultats de CI)
